# Linux WSL Handbook for beginner's
## Purpose
A practical handbook for using Linux effectively inside WSL.

This repository documents:
- WSL setup
- Common Linux + WSL issues and fixes
- Essential Linux usage
- Developer environment setup

## Who This Is For
- Beginners using WSL
- Students learning Linux
- Developers working on Windows with Linux tools

## Structure
- setup/       → WSL installation and setup
- fixes/       → Common problems and solutions
- tools/       → Git, editors, networking tools
- cheatsheets/ → Command references
- labs/        → Practice exercises
